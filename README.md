# Sistema Web de Reservas – Clínica Odontológica

Sistema web desarrollado con Python y Flask para la gestión de reservas de una clínica odontológica.

## Funcionalidades
- Registro y gestión de citas
- Panel administrativo
- Control de horarios
- Integración con Google Maps

## Tecnologías
- Python (Flask)
- PostgreSQL
- HTML, CSS, JavaScript

## Estado
Proyecto funcional. No llevado a producción por falta de información del cliente.
